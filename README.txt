To compile and run this code:

1. Please place all files in a single folder on your computer

2. Open a Terminal application

3. Move to the directory (cd) where you placed these files

4. Enter the commands:
  chmod a+x App_Installer.sh   <press enter>
  ./App_Installer.sh           <press enter>

5. After several seconds, you will be asked to enter your password

6. The Shell script will compile and test the code

7. You can run the app with

./BlogParserConsumer       <press enter>

or use:

./BlogParserConsumerStatic       <press enter>
